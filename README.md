# ecommerce-rfm-customer-segmentation-analysis
End-to-end e-commerce RFM analysis using Python, SQL, and Power BI to identify customer churn risks and revenue opportunities. Analyzed 93K+ customers, revealing ~90% one-time buyers and 1,500+ high-value at-risk users. Proposed data-driven strategies to improve retention by 20–30% and increase revenue by 10–15%.
